## 三大定律（低自由度 - 必须精确执行）

| 定律 | 规则 | 检查方式 |
|------|------|----------|
| **大纲即法律** | 严格执行大纲，不得擅自发挥 | 审查时对照大纲 |
| **设定即物理** | 实力/招式/物品 ≤ index.db 记录 | 写作前查询确认 |
| **发明需识别** | 新实体由 Data Agent 自动提取 | 章节完成后处理 |

## 新实体处理流程

当前规则：正文不再要求 XML 标签：

1. **写作时**: 直接写纯正文，新角色/地点/物品正常描写
2. **完成后**: Data Agent 自动识别新实体并写入 index.db
3. **不确定实体**: Data Agent 标记为 uncertain，由人工确认

## 章节约束分层

### Hard（必须）

- 本章可读性达标：读者能回答“发生了什么 / 谁在做什么 / 为什么”。
- 本章必须存在清晰推进：问题、目标、代价、关系变化、信息变化至少一项可被识别。
- 若上章有明确承诺（钩子/未闭合问题），本章必须回应（允许部分兑现，不要求一次性结清）。
- 禁止输出占位正文（如 `[待补充]`、`[TODO]`、`...（省略）...`）。

### Soft（建议）

- 开头尽早进入冲突/风险/强情绪（建议前 200-400 字，不再固定 120 字）。
- 未闭合问题或下一章期待锚点建议放在章末或后段（不限定“后 80-150 字”）。
- 局面变化建议保持节奏感（参考 800-1400 字一个脉冲，短章至少一次实质变化）。
- 微兑现频率按题材 profile 建议执行，不要求机械等间距。

### Style（可选强化）

- 对话尽量带意图（试探/回避/施压/诱导），减少纯说明句。
- 避免连续大段纯解释；若必须解释，建议切分为“信息 + 行动/反应”。
- 避免“回去休息了”式机械收尾；若使用平缓收尾，需同时保留未闭合期待。

## 爽点与节奏（按题材 profile 调整）

- 爽点密度由题材与章型共同决定：过渡章允许低密度，但不允许“整章无收获”。
- 组合爽点与里程碑爽点采用滚动窗口评估（5章/10-15章），用于预警而非逐章硬判。
- 连续同类型爽点达到 3 章记为风险预警，优先通过类型变体或执行差异化修正。

## Strand 平衡警告

| 情节线 | 警告条件 |
|--------|----------|
| Quest（主线） | 连续 5+ 章 |
| Fire（感情线） | >10 章未出现 |
| Constellation（世界观） | >15 章未出现 |

## 禁止事项

- `[待补充]`、`[TODO]`、`...（省略）...` → 必须完整写出
- 战斗后无善后描述（都市异能题材）

## 地图移动约束（Hard - 必须精确执行）

### 距离约束

| 移动方式 | 速度 | 一时辰行程 |
|---------|------|-----------|
| 正常步行 | 5里/时辰 | 10里 |
| 快速步行 | 8里/时辰 | 16里 |
| 普通马 | 50里/时辰 | 50里 |
| 快马 | 80里/时辰 | 80里 |
| 御剑（筑基） | 200里/时辰 | 1600里 |
| 御剑（金丹） | 500里/时辰 | 4000里 |

### 距离-时间阈值

| 场景 | 阈值 | 说明 |
|------|------|------|
| 城内短距离 | 1-2里 | 可用"片刻/呼吸" |
| 城内区域间 | 3-10里 | 需"一盏茶/小半个时辰" |
| 城郊 | 10-30里 | 需"大半个时辰/一个时辰" |
| 城际 | 30-100里 | 需"半日/一日" |
| 跨区域 | 100里以上 | 需"数日" |

### 方向约束

- 移动方向需与地图方向一致
- 禁止方向错误（如城东写成城西）
- 位置关系需与地图对应

### 时间约束

- 到达时间需与距离和移动方式匹配
- 禁止"片刻"跨越长距离

### 路径约束

- 两点间移动需有合理路径
- 禁止直接穿越障碍（山脉、湖泊、禁区）
- 需遵循道路网络

### 禁止事项

- ❌ 禁止"从东门到西门只需片刻"（距离不合理）
- ❌ 禁止"瞬间移动"除非有传送阵/瞬移技能
- ❌ 禁止方向错误
- ❌ 禁止跳过必经路径

## 道具设定约束（Hard - 必须精确执行）

### 道具属性约束

- 道具能力 ≤ index.db 记录
- 道具稀有度需与获得场景匹配
- 道具持有者需与设定一致
- 道具形态描述需与设定一致

### 道具使用约束

| 场景 | 约束 |
|------|------|
| 新道具首次使用 | 必须描写获得途径 |
| 道具能力展现 | 需与设定能力匹配 |
| 道具等级变化 | 需有明确升级描写 |

### 道具一致性检查

| 检查项 | 要求 |
|--------|------|
| 名称/别名 | 与设定完全一致 |
| 稀有度 | 与场景匹配（如凡品不出现在高阶秘境） |
| 能力效果 | 不得超出设定范围 |
| 持有者 | 不得擅自转移所有权 |
| 历史背景 | 提及需与设定一致 |

### 禁止事项

- ❌ 道具能力超出设定范围
- ❌ 道具突然出现在未获得时
- ❌ 道具稀有度与场景不匹配
- ❌ 道具所有者变更无合理理由

## 功法能力约束（Hard - 必须精确执行）

### 能力学习门槛

| 能力类型 | 最低境界要求 | 说明 |
|---------|-------------|------|
| 基础法术 | 练气期 | 简单法术 |
| 中级法术 | 筑基期 | 需要法力支撑 |
| 高级法术 | 金丹期 | 需要金丹之力 |
| 顶级秘术 | 元婴期 | 需要神识操控 |
| 阵法 | 筑基期/金丹期 | 根据阵法规模 |
| 禁术 | 金丹期/元婴期 | 通常有代价 |

### 威力大小约束

- 越阶战斗可以有，但需有充分理由（金手指、秘术爆发、克制关系等）
- 越阶击败强敌需要有伏笔（如对方有伤、擅长被克制等）
- 越阶使用法术需消耗匹配（高等法术消耗大量法力）

### 能力克制关系

| 克制类型 | 规则 |
|---------|------|
| 属性克制 | 水克火、火克木等 |
| 境界压制 | 高境界对低境界有天然压制 |
| 秘术克制 | 某些秘术有特定克制 |
| 阵法克制 | 某些阵法有特定解法 |

### 能力消耗约束

| 消耗类型 | 合理性检查 |
|---------|-----------|
| 法力消耗 | 高阶法术需高境界支撑 |
| 气血消耗 | 禁术需有代价描述 |
| 寿元消耗 | 禁术需有严重后果 |
| 冷却时间 | 强力技能需冷却 |

### 禁止事项

- ❌ 低境界无代价使用高阶法术
- ❌ 越阶战斗无合理理由
- ❌ 越阶击败敌人无伏笔
- ❌ 秘术/禁术无代价使用
- ❌ 能力凭空出现无获得途径
