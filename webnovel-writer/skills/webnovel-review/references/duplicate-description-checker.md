# duplicate-description-checker (描写重复检查器)

> **职责**: 检测外貌描写、动作描写、内心独白等在附近章节是否有重复或高度相似内容，避免"炒冷饭"和"复制粘贴"感。

> **输出格式**: 遵循统一 JSON Schema，输出结构化报告供润色步骤参考。

## 检查范围

**输入**: 单章或章节区间（如 `45` / `"45-46"`）

**输出**: 描写重复问题（外貌、动作、内心独白等）的结构化报告。

## 执行流程

### 第一步: 加载参考资料

**输入参数**:
```json
{
  "project_root": "{PROJECT_ROOT}",
  "storage_path": ".webnovel/",
  "state_file": ".webnovel/state.json",
  "chapter_range": "45-47",
  "check_window": 5
}
```

**参数说明**:
- `chapter_range`: 要检查的章节范围（如 "45" 或 "45-47"）
- `check_window`: 向前/向后检查的章节窗口数（默认 5 章）

**并行读取**:
1. 目标章节及附近章节的正文（`正文/` 目录）
2. `{project_root}/.webnovel/state.json`
3. `{project_root}/.webnovel/summaries/` 中的章节摘要（用于快速定位）

### 第二步: 描写内容提取

#### 2.1 外貌描写提取

**识别特征**:
- 角色外貌描述段落
- 包含以下关键词的句子：身高、体型、面容、眉眼、鼻梁、嘴唇、肤色、发色、衣着、服饰、气质等
- 首次出场时的详细描写

**提取方式**:
```python
# 伪代码示例
appearance_patterns = [
    r"身高.*",
    r"面容.*清秀|英俊|丑陋|普通",
    r"眉眼.*",
    r"一袭.*袍|身穿.*",
    r"气质.*",
    r"肌肤.*",
    r"肤色.*",
]
```

#### 2.2 动作描写提取

**识别特征**:
- 角色标志性动作
- 习惯性动作/手势
- 战斗中的特定招式动作
- 包含以下关键词的句子：出手、抬手、挥拳、皱眉头、眯起眼睛、冷笑、淡淡说道等

**提取方式**:
```python
# 伪代码示例
action_patterns = [
    r"皱起眉头",
    r"眯起眼睛",
    r"冷笑.*",
    r"抬手.*",
    r"身形.*闪动",
    r"身形.*晃动",
]
```

#### 2.3 内心独白提取

**识别特征**:
- 引号内的心理活动
- "他暗自想到"、"她心中暗惊"等引导词后的内容
- 包含以下关键词的句子：心想、暗想、暗自、心中、想到等

**提取方式**:
```python
# 伪代码示例
monologue_patterns = [
    r"\"[^\"]+\"",  # 引号内的内容
    r"他暗自.*",
    r"她心中.*",
    r"心中.*暗.*",
    r"暗自.*想到",
]
```

### 第三步: 相似度比对

#### 3.1 文本相似度计算

使用以下方法计算描写内容的相似度：

1. **关键词重叠法**: 计算两个文本中共同关键词的数量占比
2. **句子结构匹配法**: 匹配句子结构模式
3. **语义相似度**: 识别核心描写要素是否相同

**相似度阈值**:
| 相似度 | 判定结果 | 严重程度 |
|--------|---------|---------|
| ≥ 90% | **高度重复** | critical |
| 70%-89% | **中度重复** | high |
| 50%-69% | **轻度重复** | medium |
| < 50% | 正常 | - |

#### 3.2 检查范围

- **向前检查**: 检查当前章节前 5 章（可配置 `check_window`）
- **向后检查**: 检查当前章节后 5 章
- **同章节内**: 检查同一章节中不同位置的重复描写

### 第四步: 问题分类

#### 4.1 外貌重复

**检查内容**:
- 同一角色的外貌描写是否在前 N 章已经出现过
- 是否使用了相同或相似的外貌描写词汇

**示例**:
```
❌ 第50章: "他身高七尺，面容清秀，一袭白衣胜雪，气质不凡"
❌ 第52章: "此人身高七尺，面容清秀，一袭白衣胜雪，气质不凡"
→ 高度重复（95%），应修改第52章的描写
```

#### 4.2 动作重复

**检查内容**:
- 同一角色的标志性动作是否重复
- 战斗场景中的动作招式是否与之前章节雷同

**示例**:
```
❌ 第30章: "他冷笑一声，抬手便是一掌"
❌ 第45章: "林凡冷笑一声，抬手便是一掌"
→ 中度重复（80%），应变化描写方式
```

#### 4.3 内心独白重复

**检查内容**:
- 相似的心理活动是否重复出现
- 思考模式/分析方式是否与之前雷同

**示例**:
```
❌ 第20章: "恐怕此事另有隐情..."他暗自想到
❌ 第25章: "恐怕此事另有隐情..."他暗自想到
→ 高度重复（90%），应避免直接复制
```

## 修改建议生成指南

当发现重复描写时，生成建议应遵循以下原则：

### 外貌描写修改建议

1. **保留标志性特征，变化描述方式**
   - 原: "面容清秀，剑眉星目"
   - 改: "面目俊朗，浓眉如墨"

2. **突出不同场景下的不同特征**
   - 战斗时: 侧重气势
   - 日常时: 侧重气质

3. **使用比喻/意象替代直描**
   - 原: "一袭白衣"
   - 改: "如雪般素白的长袍"

### 动作描写修改建议

1. **变化动词**
   - 原: "冷笑一声"
   - 改: "唇角微撇"

2. **变化句式结构**
   - 原: "抬手便是一掌"
   - 改: "手掌翻飞，掌风呼啸"

3. **增加/减少细节**
   - 原: "身形闪动"
   - 改: "身形如鬼魅般闪动"

### 内心独白修改建议

1. **变化措辞**
   - 原: "恐怕此事另有隐情"
   - 改: "此事怕不是那么简单"

2. **变化引导词**
   - 原: "他暗自想到"
   - 改: "他心中不禁犯起嘀咕"

3. **变化句式**
   - 原: "恐怕...难道..."
   - 改: "难道...不成..."
