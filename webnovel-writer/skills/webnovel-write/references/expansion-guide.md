# 内容扩写指南

本文档提供章节内容扩写的指导原则，用于在章节字数不足时进行适度扩写。

## 1. 扩写基本原则

### 1.1 扩写目标

- **目标字数**：默认 2000-2500 字（关键章/高潮章可更长）
- **扩写触发**：当章节字数低于目标字数的 80% 时
- **扩写上限**：单次扩写不超过目标字数的 20%

### 1.2 扩写范围限制

**重要**：章节开头和结尾部分有承上启下的作用，**不能扩写**：

| 区域 | 说明 | 扩写 |
|------|------|------|
| **开头** | 承接上文、引入本章内容 | ❌ 禁止扩写 |
| **中间** | 本章核心内容、情节展开 | ✅ 可以扩写 |
| **结尾** | 悬念/钩子、引出下文 | ❌ 禁止扩写 |

**判断技巧**：
- **开头**：章节的前 200-400 字，通常包含承接上文的内容
- **结尾**：章节的最后 100-200 字，通常包含悬念/钩子/过渡
- **中间**：开头和结尾之间的部分，可以扩写

### 1.3 扩写原则

| 原则 | 说明 |
|------|------|
| **不增新设定** | 禁止添加新角色、新道具、新势力 |
| **不推新剧情** | 禁止添加新的情节线或重大转折 |
| **不设新冲突** | 禁止引入新的矛盾或冲突 |
| **细节补充** | 可补充已有设定的细节描写 |
| **保持一致** | 扩写内容必须与已有设定一致 |

### 1.4 允许扩写的方面

| 类型 | 说明 | 示例 |
|------|------|------|
| **风景描写** | 补充环境、氛围描写 | 天空的颜色、花草的形态 |
| **内心活动** | 丰富角色心理描写 | 犹豫、坚定、紧张等情绪 |
| **对话内容** | 扩展对话细节 | 语气、表情、动作配合 |
| **动作描写** | 细化动作过程 | 打斗细节、行动过程 |
| **感官描写** | 补充视觉/听觉/嗅觉等 | 气味、声音、温度 |
| **场景细节** | 补充场景物件 | 家具、装饰、摆设 |

## 2. 扩写方法

### 2.1 风景描写扩写

**原内容**：
```
他走在街上。
```

**扩写后**：
```
他走在青石铺就的街道上。两旁是鳞次栉比的商铺，飞檐翘角，雕梁画栋。清晨的阳光洒在瓦片上，折射出淡淡的金光。街道中央车水马龙，行人熙熙攘融，小贩的吆喝声此起彼伏。
```

### 2.2 内心活动扩写

**原内容**：
```
"我不能放弃。"他想着。
```

**扩写后**：
```
"我不能放弃。"他暗暗攥紧拳头，心中涌起一股热血。想到师尊的嘱托，想到乡亲们的期盼，那沉甸甸的责任压在肩头，让他不容退缩。纵然前路艰险，纵然敌人强大，他也必须坚持下去。
```

### 2.3 对话内容扩写

**原内容**：
```
"你来了。"他说。
"我来了。"我回答。
```

**扩写后**：
```
"你来了。"他抬起头，目光平静地注视着我，嘴角带着一丝若有若无的笑意。清晨的阳光透过窗棂，在他脸上投下斑驳的光影。
"我来了。"我点头应道，心中却有些忐忑。想到接下来的计划能否顺利执行，手指不自觉地微微颤抖。
```

### 2.4 动作描写扩写

**原内容**：
```
他挥剑砍去。
```

**扩写后**：
```
他深吸一口气体内灵气运转，周身气势骤然攀升。手中长剑一抖，挽起一朵剑花，随即身形疾动，化作一道残影向着对手斩去。剑光如虹，带着刺耳的破空声，直取对方项上人头。
```

## 3. 扩写流程

### 3.1 扩写检查清单

- [ ] 当前字数是否低于目标字数的 80%？
- [ ] **扩写是否只针对中间部分，不涉及开头和结尾？**
- [ ] 扩写是否只针对风景/内心/对话/动作/感官/场景细节？
- [ ] 扩写内容是否与已有设定一致？
- [ ] 扩写是否没有添加新角色/道具/势力？
- [ ] 扩写是否没有推动新剧情？

### 3.2 扩写执行步骤

1. **计算扩写空间**：目标字数 × 20% = 最大扩写量
2. **确定扩写范围**：
   - 开头部分（章节前 200-400 字）：❌ 不扩写
   - 结尾部分（章节后 100-200 字）：❌ 不扩写
   - 中间部分：✅ 可以扩写
3. **定位可扩写点**：在中间部分找出可扩展的段落
4. **选择扩写类型**：根据段落类型选择扩写方式
5. **执行扩写**：按原则扩写
6. **检查一致性**：确认扩写不冲突
7. **更新设定文件**：如有新细节，补充到设定文件

## 4. 设定细节补充

### 4.1 需要补充的情况

当扩写中涉及以下内容时，需要更新设定文件：

| 扩写内容 | 补充到 | 示例 |
|---------|--------|------|
| 新地貌描写 | 地图设定 | 新增山谷、河流等 |
| 角色新特征 | 角色卡 | 服饰、习惯、口头禅 |
| 道具新细节 | 道具卡 | 外观、颜色、材质 |
| 势力新信息 | 势力设定 | 新据点、新成员 |

### 4.2 补充格式

```
## 扩写补充记录

### 第{章节}章补充

**补充时间**：{时间}
**补充内容**：{描述}
**补充位置**：{设定文件路径}

详细内容：
{具体补充}
```

## 5. 禁止事项

### 5.1 绝对禁止

- ❌ 添加新角色（除非大纲明确要求）
- ❌ 添加新道具（除非大纲明确要求）
- ❌ 添加新势力
- ❌ 引入新情节线
- ❌ 改变角色性格
- ❌ 修改已有冲突的解决方式

### 5.2 高度谨慎

- ⚠️ 修改角色实力等级
- ⚠️ 改变势力关系
- ⚠️ 调整道具能力

## 6. 扩写质量标准

### 6.1 优秀扩写

- 扩写内容自然融入原文
- 细节丰富但不冗余
- 与整体风格一致
- 不影响阅读节奏

### 6.2 差劲扩写

- 生硬的填充词句
- 重复啰嗦的描写
- 与前后文风格不一
- 影响情节节奏
