# 写作规范

本文档定义了小说写作中的基本规范，用于确保文章质量并优化 token 使用。

## 1. 换段规范

### 1.1 换段原则

**核心原则**：严格控制文章换段，每一段内容应该包含完整的叙事单元。

**换段条件**（满足其一即可换段）：
1. **场景切换**：地点发生变化（如：从室内到室外、从城内向城外）
2. **时间跳转**：时间发生明显变化（如：从白天到夜晚、从今日到明日）
3. **视角切换**：叙述视角发生变化（如：从主角视角切换到配角视角）
4. **情节递进**：故事情节发生明显推进或转折
5. **信息分层**：不同类别的信息需要分开叙述（如：对话与描写分开）

### 1.2 禁止换段的情况

**以下情况不应换段**：
- 简单的一句话描述就换段
- 连续的对话没有行动描写就换段
- 同一场景内的普通描写频繁换段
- 只有一句话的过渡段落

### 1.3 换段示例

**错误示例**：
```markdown
他走进了房间。
房间很简陋。
只有一张床。
和一张桌子。
```

**正确示例**：
```markdown
他推开门走进了房间。房间很简陋，除了一张床和一张桌子，再无其他家具。墙面斑驳，像是多年未曾修缮，显然这里已经很久没有人居住。
```

**错误示例**（对话换段）：
```markdown
"你来了。"他说。
"是的。"我回答。
```

**正确示例**：
```markdown
"你来了。"他抬起头，目光平静。
"是的。"我点头应道，心中却有些忐忑。
```

### 1.4 段落长度建议

| 场景类型 | 建议字数 | 说明 |
|---------|---------|------|
| 动作场景 | 100-300字 | 快节奏，可稍短 |
| 对话场景 | 150-400字 | 对话+动作描写 |
| 心理描写 | 80-200字 | 内心活动描写 |
| 场景描写 | 100-300字 | 环境/氛围描写 |
| 过渡场景 | 50-150字 | 简洁过渡 |

### 1.5 换段检查清单

写作时检查以下问题：
- [ ] 这个段落是否包含完整的叙事信息？
- [ ] 换段是否因为场景/时间/视角变化？
- [ ] 是否有"一句话段落"需要合并？
- [ ] 段落长度是否在合理范围内？
- [ ] 连续的短段落是否可以合并？

## 2. 引号使用规范

### 2.1 必须使用引号的情况

以下内容**必须**使用引号包裹：

1. **对话**
   ```markdown
   "今天天气真好。"张三说道。
   ```

2. **内心独白**
   ```markdown
   "没想到他竟然会在这里出现。"她暗自想到。
   ```

3. **演讲/宣言**
   ```markdown
   "今日之战，有我无敌！"他高举长剑，声音洪亮。
   ```

4. **引用/书名/口号**
   ```markdown
   门上写着"浩然正气"四个大字。
   ```

5. **心理活动（直接内心独白）**
   ```markdown
   "我一定要拿到第一名！"他暗暗攥紧了拳头。
   ```

### 2.2 引号类型选择

| 类型 | 使用场景 | 示例 |
|-----|---------|------|
| 双引号「」 | 中文对话（推荐繁体中文） | 「你好」 |
| 双引号"" | 中文对话（简体中文） | "你好" |
| 单引号『』 | 引用中的引用 | 他说：「她说『再见』」 |
| 双引号"" | 内心独白/演讲 | "我要成为最强！" |

**建议**：本文档采用双引号 "" 格式，便于输入和阅读。

### 2.3 引号使用示例

**对话格式**：
```markdown
"阁下是谁？"林凡皱眉问道。
"在下青云门弟子，特来拜访。"对方抱拳道。
```

**内心独白格式**：
```markdown
"没想到竟然是他。"张三心中暗惊，"此人不是在十年前就去世了吗？"
```

**演讲/口号格式**：
```markdown
"天地不仁，以万物为刍狗！"魔君的声音响彻天地。
```

**混合使用**：
```markdown
"父亲，"他跪在地上，"孩儿不孝，无法完成您的遗愿。"
```

### 2.4 禁止省略引号的情况

以下情况**禁止**省略引号：

❌ **错误**：
```markdown
他说我今天必须去。
心理想着这很重要。
演讲的内容是大家好。
```

✅ **正确**：
```markdown
他说："我今天必须去。"
他心理想着："这真的很重要。"
演讲的内容是："大家好！"
```

### 2.5 引号使用检查清单

写作时检查以下问题：
- [ ] 所有对话是否都用引号包裹？
- [ ] 内心独白是否都有引号？
- [ ] 演讲/口号是否都有引号？
- [ ] 引用内容是否都有引号？
- [ ] 引号是否成对出现？

## 3. 规范应用指南

### 3.1 规范优先级

1. **引号使用规范**：高优先级，必须遵守
2. **换段规范**：中高优先级，建议遵守

### 3.2 润色时的检查重点

在 Step 4 润色时，应重点检查：
- [ ] 是否存在"一句话段落"？
- [ ] 换段是否有明确的原因？
- [ ] 对话是否都用了引号？
- [ ] 内心独白是否都用了引号？

### 3.3 常见问题快速修复

**换段问题**：
- 连续3段以内，每段少于30字 → 合并段落
- 同一场景无意义换段 → 删除空行/合并

**引号问题**：
- 对话无引号 → 添加引号
- 引号不成对 → 补全引号
- 内心独白未用引号 → 添加引号

## 5. 角色出场描写规范

### 5.1 首次出场描写

**首次出场角色必须重点描写**：
- 外貌特征：身高、衣着、面容特点
- 气质印象：给主角/读者的第一感觉
- 标志性特征：声音、动作、口头禅等

**非首次出场角色的描写策略**：
- **渐进式揭露**：根据角色在本章的出场方式决定描写深度
- **首次仅闻其声**：本次出场重点描写外貌
- **首次远距离印象**：本次出场可描写近距离细节
- **已有完整印象**：本次出场侧重表情、动作、心理活动

### 5.2 角色动态称呼

**称呼原则**：基于当前叙事视角动态调整对其他角色的称呼

**称呼发展阶段**：
| 阶段 | 条件 | 称呼示例 |
|------|------|---------|
| 陌生人 | 未知姓名 | 公子、姑娘、这位阁下 |
| 初识 | 仅知姓氏 | 林公子、王姑娘、李少侠 |
| 熟人 | 知道全名 | 林少侠、李小姐、张兄 |
| 亲近 | 关系友好 | 兄弟、姐妹、张兄、李兄 |
| 亲密 | 关系密切 | 昵称、小林、小李、心肝 |
| 敌对 | 关系恶劣 | 那人、那个姓林的、刺客 |
| 尊敬 | 辈分/实力高 | 前辈、掌门、长老、宗师 |

**称呼切换规则**：
- 视角切换时保持一致
- 关系变化时称呼随之变化
- 对同一角色的称呼在同一章内应保持稳定

## 6. 道具出场规范

### 6.1 道具首次出现

**首次出现道具**：必须重点描写外观
- 外观特征：形态、颜色、光泽、尺寸、材质感
- 独特标记：纹理、符文、特殊印记
- 气质感觉：给主角/读者的第一感受

### 6.2 道具首次使用

**首次使用道具**：必须重点描写功效
- 功效描述：具体效果和作用
- 使用感受：使用时的感觉（视觉、听觉、触觉）
- 限制与代价：使用限制、副作用、反噬

### 6.3 非首次出现/使用

**已使用过道具**：根据使用场景决定描写重点
- 已熟悉道具：可简写，重点放在当前使用场景的效果
- 进化/升级道具：重点描写变化之处
- 特殊场景：突出道具在当前情境下的特殊表现

## 7. 角色内心独白规范

### 7.1 内心独白格式

**必须使用引号包裹**：
```
"内心独白内容"角色动作/表情
```

### 7.2 内心独白风格匹配

若角色设置了 inner_monologue：

- **语言风格**：简洁有力/幽默/毒舌/直白/温柔等
- **内容特点**：擅长自嘲/喜欢分析/情绪化/理性冷静等
- **独白长度**：长篇大论/简短精悍/点到即止
- **句式特点**：疑问句居多/陈述句为主/感叹强烈

### 7.3 触发场景

在以下场景应使用内心独白：
- 角色内心独白 trigger_scenes 中定义的场景

### 7.4 禁忌

禁止出现：
- 角色内心独白 taboos 中定义的禁忌内容

## 8. 地图移动约束规范

### 8.1 距离约束

| 移动方式 | 速度 | 一时辰行程 |
|---------|------|-----------|
| 正常步行 | 5里/时辰 | 10里 |
| 快速步行 | 8里/时辰 | 16里 |
| 普通马 | 50里/时辰 | 50里 |
| 快马 | 80里/时辰 | 80里 |
| 御剑（筑基） | 200里/时辰 | 1600里 |
| 御剑（金丹） | 500里/时辰 | 4000里 |

### 8.2 方向约束

- 移动方向需与地图方向一致
- 禁止方向错误（如城东写成城西）
- 位置关系需与地图对应

### 8.3 时间约束

- 到达时间需与距离和移动方式匹配
- 禁止"片刻"跨越长距离

### 8.4 路径约束

- 两点间移动需有合理路径
- 禁止直接穿越障碍（山脉、湖泊、禁区）
- 需遵循道路网络

### 8.5 禁止事项

- ❌ 禁止"从东门到西门只需片刻"（距离不合理）
- ❌ 禁止"瞬间移动"除非有传送阵/瞬移技能
- ❌ 禁止方向错误
- ❌ 禁止跳过必经路径

## 情节点

**处理规则**：

- 当参考文件风格与项目整体风格差异超过 30% 时输出警告

- ❌ 禁止使用通配符批量读取

## 10. 连接词与转折词使用规范

### 10.1 核心原则

网文写作应注重语言的流畅性和节奏感，**大量使用连接词和转折词**可以：
- 增强句子的连贯性，使叙事更加流畅
- 制造情节起伏，提升阅读节奏感
- 丰富情感层次，增强代入感

### 10.2 常用连接词分类

#### 递进/承接类
| 连接词 | 用法说明 | 示例 |
|-------|---------|------|
| 就 | 表示时间紧接或因果 | 他刚坐下，**就**有人来找 |
| 然后 | 表示动作顺序 | 先敲门，**然后**推门进去 |
| 接着 | 表示承接 | 听完汇报，**接着**开始讨论 |
| 紧接着 | 表示紧跟 | 话音未落，**紧接着**一道闪电 |
| 不但 | 表示递进 | **不但**好看，**还**好吃 |
| 不仅 | 表示递进 | **不仅**是他，全家都来了 |
| 甚至 | 表示加深 | **甚至**连他自己都忘了 |

#### 转折/对比类
| 连接词 | 用法说明 | 示例 |
|-------|---------|------|
| 但/但是 | 重大转折 | 他想帮忙，**但**帮不上 |
| 然而 | 书面转折 | 情况危急，**然而**他很冷静 |
| 虽然 | 让步转折 | **虽然**很漂亮，**但**太贵了 |
| 尽管 | 让步转折 | **尽管**很努力，**还是**失败了 |
| 只不过 | 轻量转折 | 想说点什么，**只不过**没开口 |
| 没想到 | 意外转折 | **没想到**竟然是这样 |
| 哪知道 | 意外转折 | **哪知道**会发生这种事 |

#### 因果/结果类
| 连接词 | 用法说明 | 示例 |
|-------|---------|------|
| 所以 | 因果结果 | 累了，**所以**休息一下 |
| 因此 | 书面因果 | 生病了，**因此**没去 |
| 于是 | 承接结果 | 想了想，**于是**决定 |
| 也就是说 | 解释说明 | **也就是说**没有选择 |
| 这么说来 | 推断说明 | **这么说来**是他干的 |

### 10.3 使用技巧

**高频使用原则**：
- 每 100 字至少使用 2-3 个连接词
- 句子之间尽量使用连接词过渡
- 心理活动中也要使用连接词体现思考过程

**自然嵌入原则**：
- 连接词放在句首或句中，不要生硬堆积
- 根据情感强度选择合适的连接词
- 转折词可以制造悬念和期待感

**示例对比**：

❌ 较少使用连接词：
```
他走进房间。看到桌上有一封信。他打开信。信里写着要他离开。
```

✅ 大量使用连接词：
```
他推开门**就**走进房间，**看到**桌上**还**放着一封信。他走过去**然后**打开信，**没想到**信里**竟然**写着要他**立刻**离开。
```

## 11. 时态表达词使用规范

### 11.1 核心原则

中文虽然没有严格的时态变化，但**通过时态词可以清晰表达时间流动**，让读者感受到情节的推进和角色状态的变化。

### 11.2 常用时态词分类

#### 完成时态
| 时态词 | 用法说明 | 示例 |
|-------|---------|------|
| 了 | 表示动作完成或状态变化 | 他**明白了** |
| 已经 | 表示早已完成 | **已经**准备好了 |
| 曾经 | 表示过去发生 | **曾经**见过一面 |
| 早已 | 表示很早完成 | **早已**等候多时 |
| 刚才 | 表示刚发生 | **刚才**发生了什么事 |

#### 持续/进行时态
| 时态词 | 用法说明 | 示例 |
|-------|---------|------|
| 还在 | 表示仍在继续 | 人**还在**里面 |
| 仍然 | 表示状态依旧 | **仍然**不死心 |
| 依然 | 表示依旧 | **依然**相信 |
| 一直在 | 表示持续 | **一直在**等 |

#### 重复/连续时态
| 时态词 | 用法说明 | 示例 |
|-------|---------|------|
| 又 | 表示重复 | **又**来了 |
| 再 | 表示再次 | **再**说一遍 |
| 再次 | 表示又一次 | **再次**出现 |
| 重新 | 表示重新开始 | **重新**开始 |
| 接着 | 表示继续 | **接着**说 |
| 继续 | 表示不中断 | **继续**前进 |

#### 将来时态
| 时态词 | 用法说明 | 示例 |
|-------|---------|------|
| 就要 | 表示即将 | **就要**出发了 |
| 快要 | 表示接近 | **快要**到了 |
| 即将 | 表示将要 | **即将**揭晓 |
| 早晚 | 表示必然 | **早晚**会知道 |

### 11.3 使用技巧

**时态词分布原则**：
- 完成时态词（了/已经）：每段至少 1-2 个
- 持续时态词（还/仍然）：表达状态时使用
- 重复时态词（又/再）：表达连续动作时使用

**场景使用建议**：

| 场景 | 推荐时态词 |
|-----|-----------|
| 战斗场面 | 了、又、接着、紧接着 |
| 心理活动 | 了、还、又、难道、恐怕 |
| 场景转换 | 然后、接着、紧接着 |
| 情绪变化 | 了、竟然、竟然又 |
| 对话反应 | 了、原来、没想到 |

**示例对比**：

❌ 较少使用时态词：
```
他转身离开。他拿起剑。他冲过去。
```

✅ 大量使用时态词：
```
他**已经**明白了是怎么回事，**于是**转身**就**离开。**刚**走到门口，**又**听到身后传来脚步声。他**立刻**停下**然后**拿起剑，**紧接着**冲了过去。
```

## 12. 内心独白与怀疑词使用规范

### 12.1 核心原则

内心独白是展示角色思考过程的重要手段，在分析、推理、猜测时**大量使用怀疑词**可以：
- 增强角色的智慧感和思考深度
- 制造悬念，引导读者思考
- 丰富内心戏，增加代入感

### 12.2 常用怀疑词分类

#### 猜测/推断类
| 怀疑词 | 用法说明 | 示例 |
|-------|---------|------|
| 恐怕 | 担心的猜测 | **恐怕**事情没那么简单 |
| 难道 | 反问式猜测 | **难道**是他？ |
| 莫不是 | 怀疑式猜测 | **莫不是**出了什么事？ |
| 莫非要 | 推断式猜测 | **莫非要**对我不利？ |
| 該不会是 | 担忧式猜测 | **該不会**是陷阱吧？ |
| 该不是 | 怀疑式猜测 | **该不是**在骗我？ |
| 难道说 | 震惊式猜测 | **难道说**真相是这样？ |
| 难不成 | 反问式猜测 | **难不成**他还能跑了？ |

#### 不确定/疑惑类
| 怀疑词 | 用法说明 | 示例 |
|-------|---------|------|
| 也许 | 表示可能 | **也许**是这样 |
| 可能 | 表示不确定 | **可能**有危险 |
| 或许 | 表示猜测 | **或许**还有希望 |
| 说不准 | 表示不确定 | **说不准**会发生什么 |
| 说不准 | 表示不确定 | **说不准**是他 |

#### 反思/分析类
| 怀疑词 | 用法说明 | 示例 |
|-------|---------|------|
| 难道 | 反问自己 | **难道**我做错了？ |
| 真的 | 质疑确认 | 这**真的**是真的吗？ |
| 究竟 | 追问根源 | **究竟**发生了什么？ |
| 到底 | 追问结果 | **到底**想干什么？ |
| 是不是 | 自我确认 | **是不是**哪里不对？ |
| 会不会 | 担忧确认 | **会不会**有危险？ |

#### 震惊/恍然类
| 怀疑词 | 用法说明 | 示例 |
|-------|---------|------|
| 难道 | 震惊反问 | **难道**这就是真相？ |
| 原来 | 恍然明白 | **原来**是你！ |
| 竟然 | 意外震惊 | **竟然**是这样！ |
| 居然 | 意外震惊 | **居然**被他骗了！ |
| 难怪 | 恍然理解 | **难怪**总觉得不对 |

### 12.3 使用场景

#### 分析推理时
```
"此人**竟然**能在**眨眼间**突破我的防线，**难道**实力**已经**超越了我？**恐怕**事情**没那么**简单......"他眯起眼睛，**莫不是**有**什么**阴谋？
```

#### 犹豫不决时
```
"**该去**还是**不该去**？**难道**就没有**别的**办法？**或许**应该**先**试探一下？**可是****万一是**陷阱**怎么办**？"他站在原地，**迟迟**无法决断。
```

#### 震惊恍然时
```
"**原来**......**原来**一切**都**是**假的**！**难怪****难怪****难怪**他**总是**那副**表情，**竟然**被**骗了**这么久！"
```

### 12.4 使用技巧

**怀疑词使用原则**：
- 内心独白中**每段至少使用 2-3 个**怀疑词
- 连续使用**同类型**怀疑词可以加强语气
- 配合**问号**使用效果更佳
- **但不要过度使用**，否则会显得矫情

**与时态词配合**：
```
"**难道****已经**暴露了？**恐怕**他们**已经**发现了我......**竟然**会在**这种**时候**出事！"
```

**与连接词配合**：
```
"**虽然****但是**......**然而****难道**......**所以****恐怕**......"
```

## 13. 综合运用示例

### 13.1 完整段落示范

❌ 修改前（较少使用连接词/时态词/怀疑词）：
```
他听到外面有声音。他走出去看。他发现有人在打架。他出手帮忙。
```

✅ 修改后（大量使用连接词/时态词/怀疑词）：
```
他**刚**躺下，**就**听到外面**传来**一阵嘈杂声。**难道**出了什么事？他**立刻**起身**然后**走出去看。
**刚**到门口，**竟然**发现**有人**在打架。**为首**的那个**不是**昨天**刚**见过的**王五**吗？**怎么会**在这里？
"**恐怕**此事**不简单**......"他暗自想到，**但是****又****不能**不管。**于是**他**直接**冲**了**上去**......
---

### 13.2 内心独白示范

```
"**难道****真的**是**她**？**不可能**......**可是****如果不是**，**那**还有**谁****能**做到？
**恐怕****事情****远比****想象**的**复杂**......"他**皱着眉头**，**心中**却**已经**掀起了惊涛骇浪。
**原以为****只是**一件**普通**的**事，**没想到**背后**竟然**隐藏**着**如此**深的**阴谋！
**这么说来**，**之前**的**那些**事情，**也都**是**故意**的？**难怪****难怪**......
```

### 13.3 检查清单

写作时检查以下问题：
- [ ] 每 100 字是否使用了 2-3 个连接词？
- [ ] 是否有时态词表达时间推进（了、还、又、接着）？
- [ ] 内心独白中是否使用了怀疑词（难道、恐怕、难道）？
- [ ] 连续动作是否使用了承接词（然后、接着、紧接着）？
- [ ] 转折情节是否使用了转折词（但是、然而、虽然）？
