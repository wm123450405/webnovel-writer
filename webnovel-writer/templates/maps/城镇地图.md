# 城镇地图模板

> 用于创建城内布局、建筑、街道等信息

## 模板元信息

```yaml
---
map_type: 城镇地图
name: [城镇名称]
tier: 重要|次要
sub_type: 主城|巨城|大城|中城|小城
---
```

## 完整模板

```markdown
---
map_type: 城镇地图
name: {城镇名}
tier: 重要
sub_type: 大城
---

# {城镇名}

## 概述
{城镇的整体描述，包括地理位置、历史、特色}

## 基本信息

| 属性 | 内容 |
|------|------|
| 城镇等级 | 主城/巨城/大城/中城/小城 |
| 所属势力 | {势力名} |
| 人口规模 | {人口数量} |
| 地理位置 | {位置描述} |
| 特色产业 | {产业描述} |
| 建立时间 | {时间} |

## 城镇风格

### 建筑风格
- **整体风格**：{如：古色古香、庄严肃穆、繁华绚丽、朴素简约等}
- **主色调**：{如：青砖灰瓦、朱红大门、金碧辉煌等}
- **特色元素**：{如：雕梁画栋、飞檐翘角、青石路面等}
- **建筑材料**：{如：土木结构、青石建造、玉石构筑等}

### 街道风格
- **路面材质**：{如：青石板、鹅卵石、砖石等}
- **街道氛围**：{如：热闹非凡、商贾云集、清静幽雅等}
- **照明方式**：{如：灯笼、油灯、萤石灯等}

### 文化风格
- **风俗特色**：{如：崇武尚道、信奉神灵、好客豪爽等}
- **特色文化**：{如：炼器之道、炼丹之术、阵法之学等}

### 进出规则
| 规则 | 说明 | 限制 |
|------|------|------|
| 入城费 | 进城需要缴纳费用 | 每人X两银子 |

| 时间 | 城门开启时间 | 辰时开，酉时关 |

## 章节使用建议

此模板适用于：
- 主角进入新城镇时的场景描写
- 城镇内发生事件的场景设定
- 商业活动的背景设定
- 势力纷争的地理背景
